# Abbreviations

- **NFT**: Non-Fungible Token
- **PvP**: Player vs Player
- **PvE**: Player vs Environment
- **STR**: Strength (base damage)
- **SPD**: Speed (movement)
- **DEF**: Defense (mitigation)
- **STA**: Stamina
- **CDR**: Cooldown reduction (economy)
- **EIP-712**: Ethereum typed structured data signing standard
- **Escrow/Lock**: Temporary locking of an item while equipped to prevent exploit loops
- **Loadout Hash**: Hash of equipped mask + accessories + gear bound to a match
