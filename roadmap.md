# Roadmap

> High-level roadmap — subject to iteration based on playtests and security audits.

## Phase 0 — Prototype

- core combat loop
- stamina, cooldowns, knockback, and camera
- early build system with caps

## Phase 1 — Genesis launch

- Genesis Masks (fixed supply)
- Season 0 Mask Accessories
- ranked 1v1 and PvE training
- anti-exploit equip locks + signed XP claims

## Phase 2 — Seasons

- quarterly seasons (new accessories, balance patches)
- tournaments and ladders
- additional modes (2v2 / 3v3) if stable

## Phase 3 — Expansion

- new mask collections (Season 1+)
- creator tools and community events
- deeper social layer
